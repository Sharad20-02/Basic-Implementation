## Space and Time Complexity Analysis

| Algorithm                | Time Complexity | Space Complexity                             |
|--------------------------|-----------------|----------------------------------------------|
| Fisher-Yates Shuffle     | O(n)       | O(1)                                     |
| Levenshtein Distance     | O(m x n)|  O(m x n)       |
| Merge Sort               | O(n log n) | O(n)                                     |
| Soundex Algorithm        | O(n)        | O(1)                                     |
